Reference Point & Motion — Interactive Learning

Concept Overview

Motion is the change in position of an object with respect to a reference point over time.

A reference point is a fixed position we use to determine whether an object has changed its position. Without a reference point, it is impossible to say if motion has occurred.

Definitions

  • Reference Point: A stationary object or position used as a basis to measure motion.
  • Relative Motion: Motion described in relation to a chosen reference point.
  • Relative Velocity: The velocity of one object as seen from another moving object, given by vB|A = vB − vA.

Real-Life Examples

  • Inside a moving train: Trees outside appear to move backwards; another train moving alongside may appear stationary if speeds match.
  • In an airplane: Passengers appear stationary to each other, but to someone on the ground, everyone is moving at the plane's speed.
  • Walking on a moving walkway at an airport: Your speed relative to the ground is the sum of your walking speed and the walkway's speed.
